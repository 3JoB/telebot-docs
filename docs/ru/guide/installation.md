# Установить

Используйте Go Mod для установки Telebot. 

### v2
```bash
$ go get github.com/3JoB/telebot/v2
```

### v1
Версия 1 больше не поддерживается, поэтому, пожалуйста, перейдите на версию 2 как можно скорее.
```bash
$ go get github.com/3JoB/telebot
```
